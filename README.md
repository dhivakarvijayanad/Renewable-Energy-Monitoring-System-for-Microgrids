# ⚡ GridPulse – Microgrid Energy Orchestrator

GridPulse is an interactive web-based Microgrid Energy Orchestrator that simulates the generation, storage, distribution, and monitoring of renewable energy in a smart microgrid.

It provides a real-time visualization of energy flow between solar panels, wind turbines, battery storage, utility grid, and electrical loads while allowing users to control environmental conditions and observe system behavior.

---

## 🚀 Features

- ☀️ Real-time Solar Power Generation Simulation
- 🌬️ Wind Energy Generation
- 🔋 Battery Energy Storage System (ESS)
- ⚡ Utility Grid Import & Export
- 🏥 Critical & Non-Critical Load Management
- 🌐 Island Mode Simulation
- 📊 Live Power History Charts using Chart.js
- 📈 Self-Sufficiency Monitoring
- 📝 Real-Time System Activity Logs
- 🎛 Interactive Controls
  - Cloud Cover
  - Wind Speed
  - Load Demand
  - Island Mode
  - Simulation Speed
